# husen

付箋でメモできる Web サービスの作成

## 概要

ページ上に内容を書き込んだ付箋を好きな場所に貼り付けることができる．
また追加機能として，タグや複数ページを実装予定．

## 機能

### 必須

- 付箋を付ける
- 付箋を移動する
- 付箋を捨てる

### 追加予定機能

- タグ付け・タグ検索
- 複数ページ機能
- アカウント機能補完出来るように(アカウントごとにページが存在し，保管できるように)
